# 🐱 MeowUp

一只住在你桌面上的小猫咪，定时提醒你站起来活动，偶尔还会吐槽你。

## ✨ 功能

- ⏰ **智能坐站提醒** — 坐 45 分钟，站 5 分钟，到点自动提醒
- 🐱 **多猫咪品种** — 蓝白猫、黑猫、橘猫，每只猫有独立的表情和姿态
- 🖥️ **桌面宠物** — 最小化后变成桌面浮窗，陪在你身边
- 💬 **120+ 条趣味语料** — 通用吐槽、健康提醒、猫咪视角、自言自语四大类
- 🎭 **多表情切换** — 生气、悠闲、发呆，不同场景不同表情
- ☕ **休息时间** — 设置午休和下班时间，到点自动停止计时
- 🔔 **弹窗提醒** — 应用内弹窗提醒，不依赖系统通知
- 🔄 **坐站切换** — 随时切换坐姿/站姿，灵活安排
- 🪶 **轻量极速** — 安装包仅 9MB，后台安静运行
